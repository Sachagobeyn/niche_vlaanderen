Niche Vlaanderen

NICHE Vlaanderen (Nature Impact Assessment of Changes in Hydro-Ecological Systems) is a hydro-ecological model that predicts the potential occurrence of (ground)water-dependent vegetation types in an area based on information about the (abiotic) site conditions. NICHE Vlaanderen can be used to evaluate the impact of changes in the water management to groundwater-dependent vegetation. This website used to contain the full documentation for the Niche Vlaanderen package, but has been taken offline because some legal aspects were not covered. If you want to use NICHE Vlaanderen, please contact INBO.
